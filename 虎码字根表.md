## 虎码生僻字根

| 字 | 拼音 | 解释 |
|----|------|------|
| 豸 | zhì  | 古书上指没有脚的虫，也可指长脊兽；作偏旁时多见于豹、豺、貌等字。 |
| 鬲 | gé   | 古代炊器，形似鼎，三足中空；作偏旁时多与炊具、饮食器物有关。 |
| 戊 | wù   | 天干第五位；也可指斧钺一类兵器的象形来源。 |
| 弋 | yì   | 带绳的箭，用来射鸟；也有获取、巡弋之意。 |
| 臼 | jiù  | 舂米用的器具，中间凹下；也可指形似臼的部位。 |
| 殳 | shū  | 古代一种竹木制、有棱无刃的兵器；作偏旁时多与击打、动作有关。 |
| 屮 | cǎo  | 草木初生的样子；常作草、艸的本字或部件。 |
| 疋 | shū  | 脚、足的意思；另读 pǐ 时通“匹”，可作量词。 |
| 辶 | chuò | “辵”的偏旁写法，俗称走之旁；多与行走、道路、迁移有关。 |
| 隹 | zhuī | 短尾鸟的总称；作偏旁时多与鸟类有关，如雀、雁、集。 |
| 廴 | yǐn  | 长行、引导之意；作偏旁俗称建之旁，多与行走、延伸有关。 |
| 㐄 | kuà  | 跨步、两腿分开的形状；多作为字形部件使用。 |
| 禸 | róu  | 古同“蹂”，有兽足踏地、践踏之意；作偏旁时多见于禹、离等字形。 |
| 酉 | yǒu  | 地支第十位；本义为酒器，作偏旁时多与酒、发酵有关。 |
| 卩 | jié  | 古同“节”，符节、凭信之意；作偏旁俗称单耳旁。 |
| 亠 | tóu  | 汉字部首之一，俗称文字头或京字头；通常只作字形部件。 |
